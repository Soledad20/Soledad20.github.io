VER

Este proyecto está escrito en JavaScript y utiliza la biblioteca D3.js (Data-Driven Documents) para crear un gráfico de dispersión (scatter plot) interactivo. El objetivo es visualizar datos sobre el tiempo de los ciclistas que han competido en el famoso ascenso Alpe d'Huez durante varios años. Los datos se cargan desde un archivo JSON en GitHub y se procesan para que la información sea compatible con la creación del gráfico. El gráfico muestra la relación entre el tiempo de los ciclistas y el año de la carrera, y utiliza diferentes colores para resaltar los casos en los que se sospecha de dopaje. Además, el gráfico cuenta con una leyenda y una herramienta de información (tooltip) que aparece cuando se coloca el cursor del mouse sobre los puntos del gráfico.
